### 1. 本地存储功能（LocalStorage）

- **JSON 导出**：完整的数据备份格式

- 实时搜索消息内容

### 4. 标签分类系统
- 为每条消息添加多个标签
- 自动生成标签列表
- 按标签快速分类

### 5. 深色模式
- 点击右上角切换主题
- 主题偏好自动保存
- 保护眼睛，提升夜间使用体验

## 🚀 安装和使用

### 方式 1：直接打开（推荐）
1. 下载并解压 `MyWebApp-v2.0-Win.zip`
2. 双击 `dist` 文件夹中的 `index.html`
3. 在浏览器中自动打开应用

### 方式 2：使用 Python 内置服务器
```bash
cd dist
python -m http.server 8000
# 访问 http://localhost:8000
```

### 方式 3：使用 Node.js 服务器
```bash
cd dist
npx http-server -p 8000
# 访问 http://localhost:8000
```

## 🎯 快速开始

1. **添加消息**
   - 在文本框输入消息内容
   - 可选：添加标签（输入后按 Enter）
   - 点击"添加消息"或 Ctrl+Enter

2. **管理消息**
   - 搜索：输入关键词实时搜索
   - 筛选：点击标签按钮筛选
   - 删除：点击删除按钮移除消息

3. **导出数据**
   - 点击 "📄 JSON" 导出为 JSON
   - 点击 "📊 CSV" 导出为 CSV

4. **切换主题**
   - 点击右上角 🌙/☀️ 切换深色/亮色模式
