# 📱 MyWebApp - 每日数码消息收集（v2.0 增强版）

一个现代化的 Vue 3 + Vite 应用，用于收集和管理每日的数码消息、想法和灵感。

**当前版本：v2.0 增强版** - 已完成所有计划功能！🎉

## ✨ 功能特性

- 📝 **快速添加消息** - 轻松记录你的想法和消息
- 🗂️ **消息管理** - 查看、删除和清空消息列表
- 📅 **自动时间戳** - 每条消息自动记录日期和时间
- 🎨 **现代化设计** - 优美的渐变背景和流畅的动画
- 📱 **响应式布局** - 完美适配各种屏幕尺寸
- ⌨️ **快捷键支持** - Ctrl+Enter 或 Cmd+Enter 快速提交

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 开发模式
```bash
npm run dev
```
然后在浏览器中打开 `http://localhost:5173`

### 生产构建
```bash

```

### 预览构建结果
```bash

```

## 📖 使用方法

1. **添加消息** - 在输入框中输入内容，点击"添加消息"或使用快捷键
2. **添加标签** - 输入标签名称后按 Enter 或点击 + 按钮添加标签
3. **删除标签** - 点击标签上的 × 按钮删除
4. **查看消息** - 消息会按时间倒序显示在页面下方
5. **删除消息** - 点击消息卡片右下角的"删除"按钮
6. **搜索消息** - 在搜索框输入关键词，实时显示匹配的消息
7. **按标签筛选** - 点击下方的标签按钮筛选消息
8. **导出数据** - 点击 JSON 或 CSV 按钮导出数据
9. **切换主题** - 点击右上角的 🌙 或 ☀️ 切换深色/亮色模式
10. **清空所有** - 点击"清空所有消息"按钮（需要确认）

## 🎯 特色功能说明

### 消息自动排序
新添加的消息会自动出现在列表最前面，最近的消息最易被看到。

### 时间记录
每条消息都会自动记录添加时的日期和时间。

### 平滑动画
消息的添加和删除都有流畅的过渡动画效果。

### 🔖 标签分类系统
- 为每条消息添加多个标签
- 按标签快速筛选消息
- 自动生成标签列表

### 🔍 搜索和筛选
- 实时搜索消息内容
- 按标签分类筛选
- 快速定位所需消息

### 💾 本地存储（LocalStorage）
- 自动保存所有消息到浏览器本地存储
- 关闭浏览器后数据不丢失
- 支持跨会话数据恢复

### 📥 数据导出
- **JSON 格式**：适合数据备份和转移
- **CSV 格式**：适合在 Excel 等表格软件中打开

### 🌙 深色模式
- 点击右上角月亮/太阳图标切换主题
- 自动保存主题偏好
- 保护眼睛，提升夜间使用体验
