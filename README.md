# 📱 MyWebApp - 每日数码消息收集

一个现代化的 Vue 3 + Vite 应用，用于收集和管理每日的数码消息、想法和灵感。

## ✨ 功能特性

- 📝 **快速添加消息** - 轻松记录你的想法和消息
- 🗂️ **消息管理** - 查看、删除和清空消息列表
- 📅 **自动时间戳** - 每条消息自动记录日期和时间
- 🎨 **现代化设计** - 优美的渐变背景和流畅的动画
- 📱 **响应式布局** - 完美适配各种屏幕尺寸
- ⌨️ **快捷键支持** - Ctrl+Enter 或 Cmd+Enter 快速提交

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 开发模式
```bash
npm run dev
```
然后在浏览器中打开 `http://localhost:5173`

### 生产构建
```bash
npm run build
```

### 预览构建结果
```bash
npm run preview
```

## 📖 使用方法

1. **添加消息** - 在输入框中输入内容，点击"添加消息"或使用快捷键
2. **查看消息** - 消息会按时间倒序显示在页面下方
3. **删除消息** - 点击消息卡片右下角的"删除"按钮
4. **清空所有** - 点击"清空所有消息"按钮（需要确认）

## 🎯 特色功能说明

### 消息自动排序
新添加的消息会自动出现在列表最前面，最近的消息最易被看到。

### 时间记录
每条消息都会自动记录添加时的日期和时间。

### 平滑动画
消息的添加和删除都有流畅的过渡动画效果。
